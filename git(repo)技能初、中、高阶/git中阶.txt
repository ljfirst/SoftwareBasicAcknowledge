git中阶

1、git合并多个本地commit作为一次提交。
（1）git rebase -i commitId（不写commitId，默认记录与远程相差的所有commit）
（2）修改需要合并的id,修改pick为squash，退出
（3）进入commit页面，填写信息
（4）push


2、git reset --hard 后如何修复。
（1）git reflog
（2）git reset 到某个ID
（3）git log 查看恢复的commitID
（4）git checkout . 清除所有修改。


3、git cherry-pick 用法。
（1）查看分支1的git log，获取commitID
（2）切换到分支2，使用 git cherry-pick 分支1的某个commitID


4、远程库版本回退，保留部分提交的方式。
（1）git revert --no-commit commitID2..commitID4  //本次revert保留ID2，回退ID3,4
（2）git commit -m "description"
（3）git push origin master


5、远程分支跟随
（1）git branch --set-upstream-to=origin/develop develop


6、git合并其他分支
（1）git merge 分支名

7、删除远程分支
（1）方式一：git push origin --delete 远程分支名  //先删除远程
             git branch -D 本地分支名             //再删除本地
（2）方式二：git push origin :远程分支名          //先删除远程，注意origin后的空格，空格后是：
             git branch -D 本地分支名             //再删除本地
